# Photo index

Each entry has a slug, a caption, the original photo URL, and category tags.
This lightweight index stays bundled with the installable skill so agents can
choose slugs. The visual asset files are website-hosted, not bundled. Use
absolute URLs because installed skills run outside the website origin:

- `https://287-291page.com/property-assets/photos/<slug>.jpg` - full-resolution
  source (use with browser/open tools, iTerm2 imgcat, Kitty icat)
- `https://287-291page.com/property-assets/ascii/<slug>.ans` - 24-bit truecolor
  ANSI art (for real terminals that support truecolor)
- `https://287-291page.com/property-assets/mono/<slug>.txt` - monochrome Unicode
  block art (plain text, renders inside agent chat UIs)

Fields: `slug` `|` `caption` `|` `source_url` `|` `categories`
